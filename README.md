# Bot Buddy Webinars Hub

A React application for managing and registering for webinars, built with Vite and deployed on GitHub Pages.

Last deployment test: 2024-03-19

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Deployment

This project is automatically deployed to GitHub Pages using GitHub Actions. The site is available at: https://pablomacaoh.github.io/bot-buddy-webinars-hub/
